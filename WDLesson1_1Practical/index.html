Web Technologies

HTML

HTML stands for hypertext markup language. HTML is not a programming language, it is a coding language for creating and structuring web pages and their content. It uses tags and elements to define how text, images and other multimedia to appear on a webpage.

CSS

CSS stands for cascading style sheets. It is a styling language used to describe the presentaion of a document written in a markup language, most commonly HTML.
It defines how HTML elements are to be displayed on various media, such as screens, paper, or other devices.

JavaScript

JavaScript is a high-level, multi-paradigm programming language. It is primarily known for enabling interactive and dynamic content on websites.
It is one of the core technologies of the World Wide Web, alongside HTML, which provides structure, and CSS, which provides styling.
